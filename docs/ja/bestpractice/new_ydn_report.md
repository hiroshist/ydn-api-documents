# Yahoo!ディスプレイアドネットワークレポート（Ver.5.1以降）
## パフォーマンスレポートとは
パフォーマンスレポートはキャンペーン、広告グループ、広告、キーワードなどのパフォーマンスを確認するためのレポートです。<br>
レポートを表示するにはレポートやテンプレートを作成する必要があります。<br>
レポート作成の際に、レポートの種類、表示項目、集計期間などを設定することで、広告主様のニーズにあわせたカスタマイズが可能です。<br>
また、お勧めのレポート設定を利用し、簡単にレポートを作成することもできます。<br>
※こちらはYDN API Ver.5.1から有効なレポート作成方法です。<br>
<br>
広告管理ツールで提供しているレポート機能をAPIでもサポートしています。<br>

## YDN APIでの実施方法
YDN APIではレポートを表示するために、以下の2つの Serviceを使用します。  

##### 1.	ReportDefinitionService 
ReportDefinitionServiceでは、レポート出力項目の取得およびレポート定義の追加を行います。<br>
レポート定義では、レポートフィールドを指定します。  <br>

##### 2.	ReportService
ReportServiceでは、レポートの取得および作成・削除を行います。<br>
またレポートのダウンロードURLを取得する操作が提供されます。  <br>

# シナリオ
パフォーマンスレポート取得のため、YDN APIを使い以下のレポートテンプレートとレポート自体の作成から削除までのフローをご紹介します。  

#### 1.	テンプレートのフィールド情報の取得
ReportDefinitionServiceのgetReportFieldsを使用します。<br>
ReportCategoryを指定して、レポートフィールドのリストを取得します。<br>
各レポートフィールドはフィールド名、表示名（英語）、xml属性をカプセル化します。<br>

##### ＜リクエストサンプル＞（標準認証）
```xml
<?xml version="1.0" encoding="UTF-8"?>
<SOAP-ENV:Envelope
 xmlns:SOAP-ENV="http://schemas.xmlsoap.org/soap/envelope/"
 xmlns:ns1="http://im.yahooapis.jp/V5"
 xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
    <SOAP-ENV:Header>
        <ns1:RequestHeader>
            <ns1:license>1111-1111-1111-1111</ns1:license>
            <ns1:apiAccountId>2222-2222-2222-2222</ns1:apiAccountId>
            <ns1:apiAccountPassword>password</ns1:apiAccountPassword>
        </ns1:RequestHeader>
    </SOAP-ENV:Header>
    <SOAP-ENV:Body>
        <ns1:getReportFields>
            <ns1:reportType>INTEREST_CATEGORY</ns1:reportType>
            <ns1:lang>JA</ns1:lang>

```
