# RuleType (enum)
RuleTypeは、ルールタイプのリストの種類を表示します。

### Service
+ [RetargetingListService](../services/RetargetingListService.md)

| 値 | データ型 | 説明 | 
|---|---|---|
| URL| string| URLを利用したリスト設定です。 |
| LABEL| string| ラベルを利用したリスト設定です。 |
| REFERER_URL| string| リファラURLを利用したリスト設定です。 |
